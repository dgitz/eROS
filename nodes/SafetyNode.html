enumerator DEFAULT¶

Default Type, uses a ready_to_arm topic which has a bool flag and a diagnostic message

enumerator SIMPLE¶

Simple Type, just a Bool message. A diagnostic will be created if this trigger is False (not ready to arm)

Last item of list. Used for Range Checks.

class BaseNode¶
#include “BaseNode.h”

This is a BaseNode class. All Nodes should be a derived class from this Base Class.

Subclassed by DataLoggerNode, DiagnosticNode, MasterNode, SafetyNode, SnapshotNode, SystemMonitorNode

Initialize Root Diagnostic. This should be called once per instance of Diagnostic.

Parameters

  • device_name: The name of the Device. Can be hostname or any name that makes sense for the device.

  • node_name: The name of the node running that has an instance of this class.

  • system: The System identifier, uses enum: System::MainSystem

  • subsystem: The SubSystem identifier, uses enum: System::SubSystem

  • component: The Component identifier, uses enum: System::Component
